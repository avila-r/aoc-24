# Advent of Code in Elixir

Advent of Code participation using a [starter pack](https://github.com/mhanberg/advent-of-code-elixir-starter) for Elixir!

## Structure

There are 25 modules, 25 tests, and 50 mix tasks.

1. Tests filled with example solutions.
2. Solutions can be executed by `mix d01.p1`!
    - Benchmark your solution by passing the `-b` flag, `mix d01.p1 -b`

```elixir
defmodule Mix.Tasks.D01.P1 do
  use Mix.Task

  import AdventOfCode.Day01

  @shortdoc "Day 01 Part 1"
  def run(args) do
    input = AdventOfCode.Input.get!(1, 2020)

    if Enum.member?(args, "-b"),
      do: Benchee.run(%{part_1: fn -> input |> part1() end}),
      else:
        input
        |> part1()
        |> IO.inspect(label: "Part 1 Results")
  end
end
```
